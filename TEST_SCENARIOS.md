# 🧪 سناریوهای تست

این سند شامل سناریوهای تست کامل برای نرم‌افزار حسابداری است.

## ✅ سناریو 1: افزودن کالا

### مراحل:
1. برنامه را اجرا کنید
2. به بخش **کالاها** بروید
3. دکمه **➕ جدید** را کلیک کنید
4. اطلاعات زیر را وارد کنید:
   - نام کالا: `لاستیک 185/65R15`
   - برند: `میشلن`
   - موجودی: `20`
   - قیمت خرید: `5000000`
   - قیمت فروش: `6500000`
5. دکمه **💾 ذخیره** را کلیک کنید

### نتیجه مورد انتظار:
- ✓ پیام موفقیت نمایش داده شود
- ✓ کالا در لیست ظاهر شود
- ✓ تمام فیلدها پاک شوند

### نتیجه واقعی:
- [ ] Pass
- [ ] Fail

## ✅ سناریو 3: ثبت فاکتور فروش

### پیش‌نیاز:
- حداقل یک کالا وجود داشته باشد
- حداقل یک مشتری وجود داشته باشد

### مراحل:
1. به بخش **فاکتور فروش** بروید
2. مشتری را از لیست انتخاب کنید: `علی احمدی`
3. کالا را جستجو و انتخاب کنید: `لاستیک 185/65R15`
4. تعداد را وارد کنید: `4`
5. دکمه **➕ افزودن** را کلیک کنید
6. بررسی کنید:
   - قیمت کل = `4 × 6,500,000 = 26,000,000`
7. تخفیف وارد کنید: `1000000`
8. بررسی کنید:
   - مبلغ قابل پرداخت = `26,000,000 - 1,000,000 = 25,000,000`
9. دکمه **💾 ثبت فاکتور** را کلیک کنید
10. در پنجره پرسش چاپ، **بله** را انتخاب کنید

### نتیجه مورد انتظار:
- ✓ پیام موفقیت با شماره فاکتور نمایش داده شود
- ✓ پنجره پیش‌نمایش فاکتور باز شود
- ✓ تمام اطلاعات در فاکتور صحیح باشد
- ✓ لیست اقلام خالی شود

### نتیجه واقعی:
- [ ] Pass
- [ ] Fail
